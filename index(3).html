How to make a website using HTML

But First, what is HTML?

HTML (HyperText Markup Language)
is the standard language used to create and structure content on the web.
It uses tags (e.g., <h1>, <p>, <a>) to define elements like headings, paragraphs, links, and images.
It forms the backbone of all websites.

So how to create a website?
Easy, just get you laptop or computer and turn it on, open your Vis.Code(visual studio code)
If you don't have one, you can use notepad.

STEPS:

  1. Starts with basic Structure.

    Template

                    <!DOCTYPE html>
    
        <html>
    
        <head>
    
          <title>My Website</title>
    
        </head>
    
        <body>
    
          <h1>Welcome</h1>
    
          <p>This is my first website.</p>
    
        </body>
    
        </html>

  2. Now add Content

    Example:

            <P>I love creating Website!</p>
    
        <a href="https://example.com">Visit Example</a>

  3. Now add image

    Example

        <img src+"image.jpg"alt="A beautiful view">

    If you are wondering how to add image.
    First, open your folder and rename your image into something you can easily find.
    Second, put the name between this code ("").
    Third, check the website if the image is there.

  4. Add structure with List

    Example

            <ul>
    
          <li>HTML</li>
    
          <li>CSS</li>
    
          <li>JavaScript</li>
    
        </ul>

  5. Add a footer

    Example

        <Footer>
    
      <p>© 2024 My website</p>
    
    </footer>

Final Notes:

You can enhance your website further by learning CSS and JavaScript to improve its design and functionality.
